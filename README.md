##  Types of Tests Conducted

### 1. Load Test
- Simulates normal expected user traffic.
- Measures system performance under typical usage.
- Screenshot: `Load Test.jpg`

### 2. Stress Test
- Pushes the system beyond its normal capacity.
- Identifies system breaking points and stability.
- Screenshot: `Stress Test.jpg`

### 3. Spike Test
- Introduces sudden increases in traffic.
- Observes system response and recovery.
- Screenshot: `Spike Test.jpg`
